UX Friction Analysis
🔍 Current Friction Points to Discuss
Where users might drop off
These are the moments in the funnel where UX improvements could increase conversion:
  • 1. Hard gate at 2 analyses: Anonymous trial never resets. Once spent, user is permanently blocked. No "come back next month" path — it's upgrade or leave.
  • 2. Upgrade modal has no "just sign up free" option: When limit hit, modal only shows Pro ($40) and Pro+ ($100). No path to create a free account and get 2/month as authenticated user.
  • 3. Login → Workspace → Modal → Stripe = 3 redirects: User clicks "Start Pro" on results page → login page → workspace (modal auto-opens) → Stripe checkout. That's 3 page transitions before payment.
  • 4. Email capture is passive: The email form sits below results. User may never scroll to it. No modal, no timed trigger, no exit-intent.
  • 5. Free tier vs anonymous trial confusion: Anonymous gets 2 lifetime. Free authenticated gets 2/month. But there's no messaging explaining that signing up (free) gets you monthly resets.
  • 6. No trial period for Pro: No 7-day free trial or "first analysis free on Pro." User goes from $0 to $40/month with no middle step.
  • 7. Downloads work without login: Users can get the Excel + brief without ever creating an account. Reduces motivation to sign up.
