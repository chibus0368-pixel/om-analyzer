## The 4-test HBU framework

A use qualifies as HBU only if it passes all four tests in order:

1. **Legally permissible** — current zoning allows it, OR rezoning is achievable in a reasonable timeframe (12-36 mo for most jurisdictions). Check overlay districts, height limits, FAR caps, deed restrictions.
2. **Physically possible** — site can support the use given size, shape, topography, soils, utilities access, environmental constraints (wetlands, floodplain, easements).
3. **Financially feasible** — projected revenue (rent or sale) covers all costs (land + hard + soft + financing + profit) with target return.
4. **Maximally productive** — among feasible uses, this one generates the highest residual land value.

## Common HBU outcomes by site characteristic

| Site profile | Likely HBU candidates |
|---|---|
| 1-3 ac corner, retail-zoned, 25k+ ADT | QSR pad, fuel/c-store, urgent care, drive-thru bank |
| 5-10 ac, suburban, near grocery | Multi-tenant strip with grocery shadow anchor, multifamily |
| 10-30 ac, near interstate, industrial-zoned | Last-mile distribution, bulk warehouse, self-storage |
| 1-5 ac, urban infill, mixed-use overlay | 4-6 story podium multifamily over retail, hotel |
| 20+ ac, fringe, ag-zoned | Single-family subdivision, BTR (build-to-rent) community |

## Entitlement timeline rules of thumb

- **By-right development:** 3-6 months to permit
- **Site plan review only:** 6-9 months
- **Zoning variance:** 6-12 months
- **Rezoning (PUD or map amendment):** 12-24 months
- **Comprehensive plan amendment:** 18-36 months
- **Annexation + rezoning:** 24-48 months

Always pad with 25-50% for jurisdiction politics, neighborhood opposition, and required studies (traffic, environmental, archeological).

## Residual land value formula

Land value = (Stabilized NOI / Target development yield) − total improvement cost − soft costs − developer profit margin

If land is being asked at more than this number, the deal only works through density bonus, government incentives, or a higher-value use than initially modeled.

## Red flags on land deals

- Seller can't produce title commitment showing clean ownership
- Wetlands study not done OR done but not delineated
- No utilities at the property line (water/sewer extension can be 6-figure cost)
- Adjacent uses incompatible with HBU (industrial next to a retail pad)
- Phase 1 environmental report >12 months old or not done at all
- Seller demands all-cash, fast close — usually means seller knows something
